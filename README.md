# KRT Comms Rebuild — Releases

Public-Releases des Funk-Plugins für TeamSpeak 3.

**Quellcode ist privat** — dieses Repo enthält ausschließlich die fertigen
`.ts3_plugin`-Builds zum Download.

## Installation

1. Aktuellste Release-Asset herunterladen (oben rechts → Releases)
2. Doppelklick auf die `.ts3_plugin`-Datei
3. TS3-Installer durchklicken, TS3 neu starten

## Berechtigung

Plugin erfordert Mitgliedschaft in der Server-Gruppe "Kartell-Member-Rang"
auf das-kartell.org. Frag im Discord nach Freischaltung.
